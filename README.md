# Dyslexia Aware Sinhala Essay Grading System Frontend

Frontend for the Sinhala Essay Grading System that communicates with the grading API to submit essays, display feedback, and show scoring metadata.

## Features
- Essay submission form
- Grading results and feedback view
- Essay Score UI
- Visual Mindmap: Interactive mindmap visualization for essay structure analysis using Cytoscape.js

## Prerequisites
- Node.js (v18+) and npm or yarn
- Git



## Quick start (development)
1. Clone and enter the repo
   - git clone <repo-url>
   - cd sinhala-essay-grading-app-fe

2. Install dependencies
   - npm install

3. Start dev server
   - npm start

4. Open in the host browser

## Environment
Create a .env file (Use variables in .env.sample)
